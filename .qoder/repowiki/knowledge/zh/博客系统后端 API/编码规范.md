- 统一响应格式：所有 API 均返回包含 `status`、`message` 和 `data` 字段的 JSON 结构，通过 `utils.Success/Error` 系列函数封装。
- 错误码常量管理：业务错误统一定义在 `utils/errmsg` 包中，Handler 层通过状态码映射获取提示信息，避免硬编码字符串。
- 分页参数标准化：使用 `utils.ParsePageParams` 统一处理 `pagesize` 和 `pagenum`，支持 `-1` 表示查询全部并设有上限保护。
- 权限中间件链：受保护路由通过 `JwtToken()` 验证身份，管理员接口额外叠加 `AdminRequired()` 进行角色校验。
